# otalk 

webrtc capable xmpp, in the browser 

## Setup

Until we get a proper login screen, modify and save this to `localStorage.config`:

    {
        "jid": "USER@HOST",
        "server": "HOST",
        "wsURL": "wss://WSURI",
        "credentials": {
            "username": "USER",
            "password": "PASSWORD"
        }
    }
